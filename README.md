# Framr Empty Chat (TSX + Tailwind)

This project recreates the provided empty chat workspace using **Next.js App Router**, **TypeScript (TSX)**, and **Tailwind CSS**.

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this repository to GitHub.
2. Import the repo in Vercel.
3. Vercel auto-detects Next.js and deploys.
